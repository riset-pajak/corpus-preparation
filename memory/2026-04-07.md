- Output: JSONL untuk downstream processing (embedding/vector search), Markdown untuk review manusia
- Data structure: data/raw/ (sumber), data/processed/ (teks ekstrak), data/output/ (corpus.jsonl, corpus.md)
- Pipeline stages (extract/clean/structure/enrich/output) adalah bagian dari **Phase 1**, bukan "Phase 1-5" independen
